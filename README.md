# COP4331-Small Project

## **Overview**
- Web Application for COP4331 that works as a contact manager allowing login/signup and basic CRUD operations.

## **Usage**
- Login
- Sign Up
- Create Contact
- Search Contact
- Update Contact
- Delete Contact

- User authentication
- CRUD operations on contacts
- Search functionality
